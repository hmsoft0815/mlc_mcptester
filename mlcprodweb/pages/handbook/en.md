# MCP Handbook

MCP-Tester includes the complete "MCP Handbook", a structured guide for developers to master the Model Context Protocol.

## What is covered?

The handbook guides you from basic stdio deployments to advanced agentic orchestration:

- **Minimal MCP**: Setting up your first server using stdio and ping.
- **Tools, Resources & Prompts**: Understanding the core primitives of the protocol.
- **Advanced UX**: Implementing artifacts, real-time feedback, and progress monitoring.
- **Future Tech**: Deep dives into agentic servers, sampling, and human-in-the-loop elicitation.

## Quality Assurance

Chapter 14 focuses specifically on using the **Server Inspector** to maintain high standards for your MCP integrations. Use it alongside the handbook to build production-ready servers.

> You can find the full markdown source of the handbook in the `buch/` directory of the repository.
